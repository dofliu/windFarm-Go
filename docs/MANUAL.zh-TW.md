## 1. 遊戲目標與背景故事

你是一支離岸風場**運維（Operations & Maintenance, O&M）團隊**的負責人。海上的「巨神塔」風機群為全國供電，但最近故障頻傳。你的任務是：

- **調度船隊與技師、備齊備品**，在天氣窗內**登塔排除故障**。
- 把風場的**妥善率（Availability）與發電量（Generation）**維持在高檔。
- 透過**故障診斷測驗 + SOP** 學會真實 O&M 的判斷邏輯與現場限制。

主線埋藏一條懸疑線：表面上各機組各自故障，實際上背後是**跨場共用控制韌體的邏輯缺陷**，在特定海象下誤動作。隨著 7 關推進逐步揭露真相。

**核心理念**：售電才是真正的利潤來源；**故障／維修中的機組不發電**，所以「放著機組不修」會同時**少賺錢、又掉排名**——管理好機隊妥善率，就是高分玩法。

## 3. 核心工單循環（逐步說明）

> **關鍵原則：維修畫面不能直接進入。** 必須完成「出勤就緒檢查」並航行抵達機組才能維修。

```
母港接單 → 備料 → 出勤就緒檢查 → 出航 → 登塔 → SCADA 診斷答題 → SOP → 完工／大修
```

1. **接單（調度中心）**：在母港右側「待處理工單」按「接單」。先看故障的**科別**（機械/電氣/控制/結構/HSE）與**必備備品**。
2. **備料（備品交易所）**：買齊該故障的必備備品。缺貨時下單會有**到貨前置期**（要預留天數，期間可做別的事推進日）。
3. **出勤就緒檢查（出海航行畫面）**：五項缺一不可，任一 ✗ 都無法出航——
   - ☐ **船舶**可用（CTV 或 SOV）
   - ☐ **技師**對應科別、等級足夠、且**未過勞**
   - ☐ **工具**等級達需求
   - ☐ **必備備品**在庫
   - ☐ **天氣窗**允許出航（海象 ≤ 船舶耐受度）
4. **出航**：按「出航」後播放港口→風場的**俯瞰航線圖**，顯示 ETA；抵達後自動切換到「已抵達」。出航會累積**船舶磨耗**。
5. **登塔（登船）**：海象平穩可安全登船；浪高時「頂浪登船」會**延誤、耗去部分作業窗**，或選擇「返航改期」（記為安全近失、空耗 1 天）。
6. **SCADA 診斷答題**：看紅色嚴重告警＋兩個琥珀警戒提示，回答「**最先排查什麼**」4 選 1。**答錯可重作，但每次多耗作業窗**（答對 −1 時段、答錯 −3 時段）。
7. **SOP 步驟**：5 步檢核表，前兩步（確認天氣窗、LOTO 上鎖）預設完成；逐一點擊完成步驟 3~5（每步耗作業窗，工坊等級越高耗得越少）。
8. **完工**：答對診斷＋完成所有 SOP＋必備備品在庫 → 按「完工結算」。可用率回升、領獎勵、解鎖圖鑑、推進 1 天。
   - **大修（重大故障）**：發電機/主軸承級故障（`MAJOR_FAULTS`）現場拆檢完成後，轉入**多回合大修**：回母港逐日推進，需累積 **3 個「可作業天氣窗」工日**；惡劣海象會停滯且照付安裝船待命費。看準三日預報的可作業窗推進最划算。

> 維修進度會**存進遊戲狀態**：中途切換畫面不會丟失進度，作業窗也不會被免費重置。

## 4. 經濟與績效

### 預算來源（收入）
| 收入 | 說明 |
|---|---|
| **售電收入（主要現金流）** | 每推進一天 = **實際運轉機組發電量 × 售電單價（3,200 ◎/MWh）**。故障／維修中的機組不計入。開局約 31 萬/日（Hub「售電收入/日」） |
| 工單報酬 | 完成主線/工單的獎勵（設計為「涵蓋所耗備品＋小額里程碑」，避免與恢復發電的收入重複計） |
| 戰情室修復報酬 | 每在戰情室修復一台機組 +12 萬 |
| 例行小任務 | 調度中心，穩定累積 |
| （選）備品價差 | 交易所買低賣高 |

### 支出
| 支出 | 數值（約） |
|---|---|
| 備品採購 | 耗材數萬 ~ 大型組件近千萬（葉片 900 萬） |
| 倉儲維持費 | 每件備品 2,000 ◎/日；另有每日折舊報廢機率 |
| 技師薪資 | 每日計薪（基本 2,200 + 等級×700 /日，月薪約 9~13 萬）；按在職天數自動分攤，解僱即停 |
| 出海動員費 | 每趟 25 萬（同一趟同時修多台可分攤 → 鼓勵批次） |
| 升級/保養 | 設施升級 (等級+1)×100 萬；購置 SOV 3,000 萬；進階檢測 800 萬；船舶保養 200 萬 |
| 停機損失 | 工單未修每天約 3 萬（戰情室停機則直接折抵當日淨發電） |
| 安裝船待命費 | 大修惡劣海象停滯，40 萬/天 |
| **SLA 違約金** | 季度平均可用率 < 90% → 500 萬 |

> ⏱ **單一時間軸**：幾乎所有動作都會推進天數（維修出勤、招募、升級、保養、推進大修、遠端巡檢、戰情室推進、靠港休整…）；採購則以「到貨前置期」表現時間成本。
> 💡 **莉莉財報**：每推進一天，母港「今日收支」會列出當日各項現金流（售電/修復報酬/薪資/倉儲/停機/待命/SLA/事件 → 淨額），幫你看懂錢的去向。

### 綜合績效分（排行榜，白話版）
```
綜合績效分 =  發電量(淨值)
           + 可用率 × 5
           + 完成任務 × 30
           − 安全事件 × 20
           + 營運風場數 × 10
           − SLA 違約次數 × 25
           + 戰情室修復台數 × 8
           − 累積停機損失 × 0.3
```
白話：**多發電、把妥善率拉高、多完成任務、別出安全事件、別違約、勤修機組**就是高分；**放著機組停機**會從「少賺發電」與「停機損失扣分」兩頭傷害你。

### 如何衝高分（小技巧）
1. **早期穩現金**：用調度中心例行小任務＋遠端巡檢累積預算與經驗。
2. **招齊各科別技師**：不同故障需不同科別；技師有限時戰情室要排優先序。但別過度招募（薪資會吃掉利潤，必要時解僱）。
3. **積極管理戰情室**：故障台數越多、停機越久越虧；同一趟出海同時修多台分攤動員費。
4. **看三日預報排程**：預知風暴 → 提前搶修或靠港等更好的天氣窗；大修務必排在可作業窗。
5. **守住季度 SLA**：平均可用率別跌破 90%，否則季末罰 500 萬＋扣分。
6. **JIT vs 安全庫存**：庫存越多停機風險越低，但倉儲費與折舊越高——抓平衡。

## 5. 進階系統

| 系統 | 重點 |
|---|---|
| **技師疲勞** | 出勤（維修/大修/定檢）累積疲勞，達輪班上限（80）**不得派工**；靠港休整每日回復。需招同科別技師輪班。 |
| **船舶磨耗與保養** | 每趟出航累積磨耗，過高（≥55/85）會**縮短維修作業窗**；於 CTV 整備廠付 200 萬保養歸零。 |
| **備品在途與倉儲折舊** | 缺貨下單有到貨前置期；持有備品每日課倉儲維持費，且每類每日有折舊報廢機率。 |
| **天氣窗與三日預報** | 天氣是時間軸屬性（馬可夫慣性，風暴成群），含約 18% 預報誤差；母港/出航/HUD 皆顯示三日預報＋風暴警示，支援預防性排程。 |
| **多回合大修** | 重大故障拆檢後轉入大修，需 3 個可作業天氣窗工日；惡劣海象停滯並付待命費。 |
| **突發事件** | 日推進依機率觸發（人員短缺、罷工、交貨延遲、定期大修、突發故障、對手搶單、韌體更新、風平浪靜…）；健康度越低越常觸發且偏壞事件。 |
| **合約 SLA** | 每季 90 天結算平均可用率，< 90% 罰違約金並扣分。 |
| **機組健康度** | 隱藏值（0-100），每日衰退、未修故障加速劣化 → 提高連鎖故障機率；維修/休整/良好預防可回復。 |
| **多風場拓展** | 達資金＋資歷門檻可拓展彰化/雲林/苗栗/澎湖共 4 座，每座 24 機組、不同海域背景與每日發電。 |
| **自由營運沙盒** | 自由營運中心的無限判斷型任務，永遠開放、只進排行榜；含即時動畫輔助圖，付費「進階檢測」後判讀更清晰。 |

### 風場戰情室（即時機隊營運層）
這是與主線/沙盒並存的即時層，對應真實 O&M 的「同時多工」：
- **機組個體**：每座風場 24 台，各有狀態（正常/故障/維修中）與發電佔比；開局即有數台故障。
- **派工 Dispatch**：指派對應科別、未過勞、未在工的技師維修故障機組（消耗對應備品；現場同時工單數受**船舶上限**限制：CTV 2 / SOV 4 / 每整備等級 +1）。
- **遠端重啟 Reset**：軟性故障（偏航/變流器/感測器誤報）可**免技師、1 天**遠端清除，讓有限人力專注硬故障。
- **預防性定檢 Inspect**：派一組人巡檢（2 天）→ 後續一段時間故障率下降。
- **推進一天 Advance**：與全域時間軸共用（天氣/SLA/疲勞/倉儲一起推進）。

## 7. 故障題庫（教學重點，共 5 種）

| 故障 | 科別 | 必備備品 | 「最先排查」正解 | 知識點 |
|---|---|---|---|---|
| 齒輪箱油溫過高 | 機械 | 齒輪箱齒輪油 | 潤滑油油位與油質 | 潤滑/散熱 |
| 偏航失準 | 控制 | 偏航電機 | 偏航煞車未完全釋放 | 煞車/齒盤 |
| 發電機振動異常 | 機械 | 變槳軸承 | 軸承內環故障（BPFI 特徵頻率） | 振動頻譜 |
| 變槳系統故障 | 控制 | 液壓油 | 變槳後備電池/超級電容 | 安全停機 |
| 變流器過溫跳脫 | 電氣 | 變流器模組 | 冷卻流量不足/堵塞 | IGBT 散熱 |

> 發電機振動為**重大故障**，需走多回合大修流程。

## 8. 常見問題（FAQ）與小技巧

**Q：為什麼「出航」按鈕是灰的？**
A：出勤就緒檢查有項目未過（紅色 ✗）。常見是缺備品、沒有對應科別技師（或技師過勞）、或海象超過船舶耐受度。依提示去對應設施準備。

**Q：海象「停航」怎麼辦？**
A：CTV 只能到「警戒」、SOV 可到「停航」。可「靠港休整」推進一天重新評估海象、購置 SOV，或看三日預報等下一個可作業窗。

**Q：作業窗關閉、被迫撤離了？**
A：答錯多耗時段、每步驟也耗時段；窗歸零就得撤離（記安全近失）。對策：升級工具（降每步耗時）、升級船隊（加作業窗）、別在浪高時硬上、診斷先想清楚再答。

**Q：現金越來越少？**
A：確認售電收入＞開銷。停機機組越多越虧；別過度囤備品（倉儲費）或過度招募（薪資）；積極在戰情室修復機組恢復發電。

**Q：主線下一關打不開？**
A：計分主線**按週開放**（教師設定）。本週任務完成後若下一關屬下週會鎖定——先去自由營運中心衝排行榜。

**Q：進度會存嗎？**
A：以 localStorage 自動存檔；登入後績效分會上傳雲端排行榜（可選完整存檔同步）。
